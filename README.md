#smart event dashboard:

A modern Event Management Dashboard built using HTML, CSS, and JavaScript that allows users to create, view, and manage events dynamically using DOM manipulation.

This project focuses on practicing:

-DOM selection and manipulation
-Event handling (mouse + keyboard)
-Dynamic UI updates
-Form handling and validation
-Clean UI styling with modern CSS

Features:

-Add events using a form
-Delete individual events with smooth animations
-Add sample events instantly
-Clear all events with confirmation dialog
-Smart empty-state handling
-Live DOM event tracker (keyboard + mouse demo)
-Responsive card-based layout

Project Structure:
smart-event-dashboard/
│
├── index.html      # Main layout
├── style.css       # UI styling
├── script.js       # DOM logic & events
└── README.md


Tech Stack:

HTML5 — Semantic structure
CSS3 — Modern styling, grid layout, transitions
JavaScript — DOM manipulation & logic
